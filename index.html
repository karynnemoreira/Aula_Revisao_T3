O que é lógica de programação?

É definir os passos e a ordem necessária para alcançar um objetivo.

O que é o Java Script?

É uma linguagem de programação que tem o intuito de deixar a página mais dinâmica.

Escopo Global

O escopo global é quando uma variável é declarada FORA DE UMA FUNÇÃO.

Escopo Local

O escopo local é quando uma variável é declarada DENTRO DE UMA FUNÇÃO.

Variável

Uma caixinha onde armazenamos informações

  • VAR: *Podem ser atualizadas e declaradas novamente dentro de seu escopo.
  • LET: *Podem ser atualizadas, mas não podem ser declaradas novamente.
  • CONST: *Não pode ser atualizado e nem declarado novamente.
